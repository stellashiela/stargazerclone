(Text Formatting)

Javascript command to display text to console is console.log(). We can also display the variable content with this function. For example, if we have the variable name, then the program code can be written like this: 

                    var name = "StarGazer";
                    console.log(name);
So the outpot is: StarGazer

To rerun the program, refresh it by pressing the F5 button.
